# My build of dwm

dwm with a few patches added to it:

- Fullscreen with `Mod-Shift-F`
- Cycle layouts with `Mod-Ctrl-,` and `Mod-Ctrl-.`
- Hide vacent tags
- Scratchpad with "Mod-`"
- Stacker
- Clickable modules with dwmblocks
- Sticky windows across tags with `Mod-s`
- Window swallowing
- Vanity gaps
- Attach bottom
- Always center

## Install

```
cd dwm
sudo make install
```
